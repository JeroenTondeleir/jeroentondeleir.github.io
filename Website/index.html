Eerste keuze

Voor mijn eerste statische site generator heb ik hexo gekozen.
Ik heb hiervoor gekozen omdat het een makkelijke static site generator is, maar ook omdat ik moeilijkheden heb met keuze’s maken.

Hoewel hexo oorspronkelijk werkt met markdown, kan je er toch nog html meteen in verwerken, dit zorgt ervoor dat je bepaalde interesante combinatie’s kan doen.
Ik ben op het einde van het jaar nog steeds geen expert met hexo, maar ik weet toch al een groot deel. Hoewel dat markdown zeer beperkt vond ik het zeer interessant dat je er veel mee kon doen.
