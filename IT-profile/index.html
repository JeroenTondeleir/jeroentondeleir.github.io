IT-profile

Hier kunt u een beeld vormen van mijn vaardigheid in een bepaalde programmeertaal.
Deze cijfers worden weergeven op een schaal van nul tot vijf.

Legende

0: ik kan/begrijp er niks van.
1: ik begrijp de taal, maar ik kan hem moeilijk gebruiken.
2: deze taal kan ik gebruiken a.d.h.v. internet, boeken en andere documentatie.
3: deze taal kan ik gebruiken op een beginnersniveau.
4: ik kan deze taal tamelijk goed.
5: ik kan deze taal zéér goed.
