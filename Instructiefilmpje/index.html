Ondertiteling van een instructiefilmpje

De opdracht bestond om een filmpje te zoeken op YouTube en dit in het Nederlands te ondertitelen a.d.h.v. zelfgezochte tools.

De link van het filmpje: https://www.youtube.com/watch?v=4ZCOtvPqDBY
Het originele filmpje: https://www.youtube.com/watch?v=DRijgLlplVw

Werkwijze

Eerst heb ik op internet gezocht naar een bepaalde site waarmee ik mijn YouTube filmpje kon downloaden. Deze site werd keepvid.
Nadien ben ik op zoek gegaan naar een tool waarmee ik de ondertiteling kon opschrijven wanneer de tekst vermeld werd in het filmpje.
Dit heb ik gedaan aan de hand van Aegisub.
Nadat dit klaar was heb ik het geconverteerd naar een .srt bestand, die functie zit al in het programma.
Als laatste heb ik het programma AVC (Any Video Converter) gebruikt.
AVC laat je toe om een video van elk formaat naar een ander formaat te formateren met een paar extra opties erbij, ik had die tool nodig om mijn ondertitels bij mijn filmpje te voegen.
Nadien heb ik mijn filmpje op youtube geüpload zodat iedereen mijn werkje kan zien.
