Case-study

Inleiding

Mijn opdracht was om een case-study te maken over VPN.
Mijn taak bestond eruit om een goedkopere versie te vinden van vpn-software zodat AZ Herentals deze kunnen gebruiken
AZ Herentals maakt nu gebruik van checkpoint, wegens compabiliteitsproblemen moeten ze sommige software dubbel aankopen zodat ze alles kunnen gebruiken, zowel thuis als op het werk..
Deze tekst is opgebouwd uit verschillende software-programma’s, die ik ga vergelijken met elkaar.

Wat is VPN?

VPN is de afkorting van Virtual Private Network.
Een VPN is verbonden door geëncapsuleerde protocolen door een WAN (Wide Area Network), zoals het internet. Dit betekent dat het netwerk zich niet moet bevinden op één fysieke locatie zoals LAN (Local Area Network).
Daardoor is het mogelijk om vanuit thuis met je computer op het werk kan vinden

Software van VPN

Cisco VPN Client

Software ontwikkeld door Cisco, in 2014 eindigde de EOL (End-Of-Life).
Het is niet zeker of de software nog bestaat, maar hij wordt niet meer bijgewerkt.
Cisco suggesteert op hun website om te migreren naar Cisco Anyconnect Secure Mobility Client.
De software is gratis te downloaden, maar je moet wel apparatuur van Cisco hebben.
Om het te downloaden moet je geregistreerd zijn op de site en dat lukt enkel als je software hebt aangekocht.
Er is zeer duidelijk vermeld dat de software niet compatibel is met apparatuur die niet van Cisco is.

OpenVPN

Dit is een Opensource project, dus dat betekent dat het gratis is.
OpenVPN maakt gebruik van de SSL (Secure Sockets Layer) technologie voor hun netwerkveiligheid.
Als men via OpenVPN een VPN connectie wil maken, dan moet er een OpenVPN server geïnstaleerd zijn en de configuratiegegevens moeten beschikbaar zijn voor de clients. Origineel bestaat de software uit cmd (Command prompt) applicaties voor de client en server, maar hiervoor bestaat er een ander programma om het gebruiksvriendelijker te maken.
De programma’s zorgen ervoor dat het een GUI (Graphical User Interface) krijgt.
Zo heeft men voor de clients het programma: ‘OpenVPN GUI’ en voor de server: ‘OpenVPN Control’.

Windows Built-in VPN

Dit programma staat standaard op je windows pc, maar daarentegen heeft het wel een paar nadelen.
Je moet de bevoegdheid hebben om de poorten van je router te kunnen openzetten.
Je moet je windows en een PPTP (Point-to-Point Tunenling Protocol) poort openzetten voor het internet, dit is niet ideaal voor de veiligheid.
Het is niet zo gemakkelijk om ermee te starten en qua gebruiksvriendelijkheid zijn er betere alternatieven (zoals LogMeln Hamachi).
Door deze nadelen, vind ik dit geen aanrader.

Besluit

Als ik alles met elkaar vergelijk, kom ik als oplossing OpenVPN uit.
Voornamelijk omdat deze goed is qua veiligheid (zeker tegenover de Windows Built-in VPN), maar ook omdat hier geen specifieke hardware voor nodig is.
Als Cisco compatibel was met alle hardware zou dit zeker mijn eerste keuze zijn.
Bedankt voor het lezen en ik hoop dat u het informatief vond.
